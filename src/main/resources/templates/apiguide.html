How to use REST API

This is a simple guide on how to use the REST API functions of this web app. The purpose of the API is to return the team composition and calculations in JSON format. For starters, the API can be accessed by using the link below:

pokemontrainingtool.up.railway.app/api/team

At the end of the url, add another slash followed by the team name you want to access, provided it exists, e.g pokemontrainingtool.up.railway.app/api/team/teamname

If you wish to generate a team from scratch directly via the API, you can too! Simply use the link below and enter your Pokemon names, separating each Pokemon with "%2b".

pokemontrainingtool.up.railway.app/api/generate?team=
